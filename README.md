# TheTest

Простой адаптивный сайт для подготовки к тестам (GitHub Pages).

## Данные

Вопросы загружаются из файла `/tmp/workspace/denis-gr/TheTest/data.jsonl` в формате JSONL:

```json
{"tx":"Текст вопроса","a0":"Вариант 1","a1":"Вариант 2","a2":"Вариант 3","a3":"Вариант 4","cr":"0"}
```

Где:
- `tx` — вопрос
- `a0..a3` — варианты ответа
- `cr` — индекс правильного ответа (`0..3`)

## Поведение сайта

- Загружает вопросы через `fetch` из `data.jsonl`
- Не показывает уже отвеченные вопросы повторно (с использованием `localStorage`)
- Позволяет показать ответы в формате:
  `[{ "tx": "...", "sc": "0" }]`
- Позволяет очистить локальное хранилище кнопкой «Сбросить прогресс»
